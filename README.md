# gas-ts-template

Google Apps Script (TypeScript) Template.

## install

```
npm install
```

## login

```
npm run login
```

If never used Google Apps Script before, go to <https://script.google.com/home/usersettings> and Enable "Google Apps Script API".

## create

```
npm run create
```

## clone

```
npx clasp clone --rootDir="./dist" <Project URL>
````

## open

```
npm run open
```

## deploy

```
npm run deploy
```
